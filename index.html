Теги HTML

  • <h1>...</h1> - заголовок первого уровня
  • <p>...</p> - абзац
  • <a href="...">...</a> - ссылка
  • <img src="..." alt="..."> - изображение
  • <div>...</div> - контейнер
  • <span>...</span> - встроенный контейнер
  • <ul> <li>...</li> </ul> - ненумерованный список
  • <ol> <li>...</li> </ol> - нумерованный список
  • <table>...</table> - таблица

Свойства CSS

  • color - цвет текста
  • background-color - цвет фона
  • font-size - размер шрифта
  • margin - внешние отступы
  • padding - внутренние отступы
  • border - граница элемента
  • display - тип отображения элемента
  • flex - гибкое выравнивание элементов
  • grid - сеточная раскладка

Примеры

Пример HTML


<!DOCTYPE html>
<html>
<head>
    <title>Мой сайт</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это пример HTML.</p>
</body>
</html>

Пример CSS


body {
    background-color: #f3e5f5;
    color: #4a148c;
    font-family: Arial, sans-serif;
}

h1 {
    font-size: 2em;
    text-align: center;
}

p {
    font-size: 1em;
    margin: 10px 0;
}
